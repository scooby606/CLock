# CLock - STM32 Engineezy Tipping Marble Clock

## Hardware Target

- STM32 NUCLEO-F103RB
- DRV8825 stepper driver
- MMOBIEL 17HS4023 NEMA17 stepper motor
- Engineezy Tipping Marble Clock
- DS3231 RTC

## Firmware Goals

- Smooth stepper motion control
- Accurate RTC based timing
- Marble lift control
- Calibration routines
- USB diagnostics
